# Personal Portfolio

A sleek, responsive, and minimalistic personal portfolio designed to showcase digital products, experience, and development skills. 

## Features
- **Dark Mode Aesthetic**: A clean, monochromatic slate and charcoal palette with high contrast for readability.
- **Responsive Design**: Fluid and mobile-friendly layout across all screen sizes.
- **Vanilla Tech Stack**: Built entirely with HTML5, CSS3, and Vanilla JavaScript—no heavy frameworks or dependencies.
- **Micro-Animations**: Uses modern CSS transitions and intersection observer APIs to provide a dynamic feeling and smooth reveal animations.
- **Fast Performance**: Semantic HTML and optimized asset sizes ensure fast loading and optimal browser performance.

## Hosting & Deployment
This repository is configured out-of-the-box to be deployed using **GitHub Pages**. A Github Action workflow (`.github/workflows/static.yml`) is included to automatically compile and distribute the project whenever changes are pushed to the `main` or `master` branch.

### Viewing Locally
To view this project locally without a web server:
1. Clone or download the repository.
2. Open the `index.html` file directly in your web browser.

Alternatively, developers can use a local static server like `vs code Live Server` or `npx serve`.
